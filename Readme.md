## RAG_Technique

This repository contains implementations of various Retrieval-Augmented Generation (RAG) techniques for experimenting with different retrieval and fusion strategies using Python and Jupyter notebooks.

## Setup for Mac
1. python3 -m venv myenv
2. virtualenv myenv
3. source myenv/bin/activate
4. pip install jupyter notebook ipykernel
5. python3 -m notebook


This will create a folder named myenv containing the virtual environment.
